
Scripts Python développés dans le cadre d'un apprentissage orienté freelance (automatisation, scripts, bots).

## Projets

- **calculatrice.py** — Calculatrice avec 4 opérations et gestion d'erreurs
- **temperature.py** — Convertisseur Celsius/Fahrenheit
- **generateur_mot_de_passe.py** — Générateur de mot de passe aléatoire
- **journal_de_contacts.py** — Carnet de contacts avec stockage en liste
- **gestionnaire_de_notes.py** — Lecture et écriture de fichiers texte

## Comment lancer un script

